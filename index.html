OS Fins justificam os meios?

A suprema obrigação do governante é manter o poder e a segurança do país que governa, ainda que para isso ele tenha que derramar sangue.

se os objetivos forem importantes o suficiente, qualquer método para os atingir é aceitável.

Os fins justificam os meios.

significa que não importa qual foi o caminho tomado, desde que o resultado seja vitorioso.
